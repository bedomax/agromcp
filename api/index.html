AgroMCP

Model Context Protocol for Agricultural Data

Un servidor MCP de referencia que expone datos agrícolas a través de endpoints REST simples. Diseñado para conectar información de operaciones agrícolas con modelos de IA.

Endpoints

GET /api
Información de la API y lista de endpoints disponibles
GET /api/labors
Labores y actividades agrícolas realizadas en el campo
GET /api/costs
Registros de costos operacionales y gastos de producción
GET /api/workers
Información sobre personal y trabajadores del campo
GET /api/crops
Datos sobre cultivos, variedades y ciclos de producción
GET /api/health
Estado del servidor y disponibilidad del API
